# ⚡ CYBER POMODORO // 赛博番茄钟 ⚡

> **"NEURO-ENHANCED FOCUS PROTOCOL"**
> 
> 这是一个拥有纯正赛博朋克美学的交互式番茄钟。专为在霓虹灯下工作的开发者与极客设计。


---

## 🛰️ 核心特性 // FEATURES

- **视觉冲击**：动态霓虹渐变背景，配合 Orbitron 数字科技字体。
- **故障艺术 (Glitch Art)**：标题具备动态故障抖动效果，增强沉浸感。
- **CRT 扫描线**：模拟老式显示器的扫描线图层，还原复古未来感。
- **神经协议交互**：
  - `INITIATE` (开始): 激活聚焦协议。
  - `SUSPEND` (暂停): 挂起当前任务流。
  - `RESET` (重置): 初始化时间参数。
- **智能提醒**：计时结束自动触发浏览器级别通知提醒。
- **快捷指令**：
  - `Space`: 快速开始/暂停。
  - `R`: 快速重置系统。
